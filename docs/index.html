Предисловие

Здравствуйте, этот «учебник» для тех, кто хочет научиться писать ботов для Telegram с использованием библиотек pyTelegramBotAPI (Уроки 1-12) и aiogram (Урок 13 и новее). Автор не ставил собой целью сделать всеобъемлющую документацию, а лишь познакомить читателя с прекрасным миром ботов.
Новичкам рекомендуется – после ознакомления с языком Python, разумеется, – читать уроки по порядку, более продвинутые разработчики вольны использовать «учебник» в режиме справочника.

Связаться с автором можно по адресу groosha (собака) protonmail.com.
Все исходные коды к урокам расположены в этом репозитории, а обсудить материал можно в Telegram-чате.
